# Taller02-Frontend

Landing desarrollada con **React + Vite** para el Taller 02.

## Requisitos

- Node.js 18 o superior
- npm 9 o superior

## Instalación

1. Abre una terminal en la carpeta del proyecto.
2. Instala dependencias:

```bash
npm install
```

## Ejecutar en modo desarrollo

```bash
npm run dev
```

Al iniciar, Vite mostrará una URL como `http://localhost:5173`.

## Ejecutar desde tu IDE (paso a paso)

### Visual Studio Code

1. Abre VS Code.
2. Ve a **File > Open Folder** y selecciona `Taller02-Frontend`.
3. Abre la terminal integrada con **Terminal > New Terminal**.
4. Ejecuta:

```bash
npm install
npm run dev
```

5. Haz `Ctrl + click` sobre la URL que aparece en terminal para abrir la app en el navegador.

### WebStorm / IntelliJ

1. Abre el proyecto desde **Open**.
2. Abre la terminal del IDE.
3. Ejecuta:

```bash
npm install
npm run dev
```

4. Abre la URL local mostrada por Vite.

## Build de producción

```bash
npm run build
```

La carpeta de salida será `dist/`.

## Vista previa del build

```bash
npm run preview
```
